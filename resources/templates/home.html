Magic Wiki Templator

This app allows you to easily fill in templates kept in Github wiki pages. It was written for ClojureBridge Berlin to for filling in email templates.

Step 1. Create a page on a Github wiki, for example this one.

Step 2. Use placeholders on the page, like 

Hello {{first_name}}
You can also provide a default value to fill in, like 
Thanks, {{sender_name|The ClojureBridge Berlin Team}}

Step 3. Add this snippet to the page: 

:memo: [FILL IN THIS TEMPLATE](https://wiki-templator.herokuapp.com/fill-in) :memo:

Step 4. Profit.
